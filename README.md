# Springboot 自习室预约小程序

## 核心功能
- 用户预约自习室
- 管理员管理自习室资源
- 实时查看自习室使用情况
- 预约提醒功能

## 适用场景
- 学校学生自习需求
- 办公室职员学习交流
- 公共图书馆自习空间管理

## 技术特点
- 基于 Spring Boot 和 Vue.js 的前后端分离架构，提高开发效率和用户体验
- 实时数据同步，确保自习室预约信息的准确性
- 简洁的用户界面，易于操作和使用
